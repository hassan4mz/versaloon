About
=====

JTAG Versaloon firmware for the STM32 Bluepill board.

Build
=====

If you have docker, you can just run ```./build.sh```, it will build the firmware for you.

Binaries
========

Binaries are in the directory ```release/firmware/STM32F103C8_BluePill```.
